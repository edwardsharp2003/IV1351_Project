# IV1351\_Project

## Scripts

To run the scripts, follow the installation proccess of the required dependencies inside of an venv

1. Install python3 on your machine 

2. Install the venv directory 
```
python3 -m venv venv 
```

It's important to use venv instead of the standard .venv 

2. Run the venv script to start it 

For regular zsh terminal
```
source venv/bin/activate
```
For the fish terminal 
```
source venv/bin/activate,fish
```

3. install the faker dependency
```
python -m pip install faker 
```

4. exit the venv 
```
deactivate
```
